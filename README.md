# Image Generator

Skillcrush practice exercise: fetch a random image array.

When the button is clicked, a random image will appear on the page, including the photographer’s name! 🖼️

### Skills practiced

- APIs
- async function
- fetch()
- await 
- json()
- Math.random()
- Math.floor()
